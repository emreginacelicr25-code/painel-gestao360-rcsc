# Painel de Gestão RCSC

Plataforma de gestão escolar da E.M. Regina Celi da Silva Cerdeira — painel do diretor,
Crescente (tarefas), Busca Ativa, Projetos & Ações, Equipe, Legislação & Documentos,
Indicadores e integração com as demais plataformas do ecossistema (rcsc-triagem,
transporte-escolar, plataforma-inclusao-rcsc, central-documentos-rcsc).

Construída em etapas — veja `docs/PROGRESSO.md` para o estado atual e os próximos passos,
e `docs/DEPLOY.md` para publicar no Supabase + Vercel.

## Rodando localmente

```
npm install
cp .env.example .env   # preencha com suas credenciais Supabase
npm run dev
```
